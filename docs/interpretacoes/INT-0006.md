# INT-0006 — Q-MOBILIZACAO-FATO-GERADOR

**Dispositivo:** LC 214/2025, art. 254
**Status:** proposta
**Autor:** registro inicial do projeto

**Perfil sugerido:** conservador

## Questão

Mobilização e canteiro: quando ocorre o fornecimento do serviço?

## Posição defendida

O fato gerador ocorre a cada evento de fornecimento contratado (medição), incluindo mobilização quando prevista como item de pagamento

## Efeito no cálculo

`fato_gerador=medicao`
